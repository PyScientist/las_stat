las_stat v1.1 получение полезной информации для файла либо группы файлов las

Структура:

razbor_files_in_folder.py
на вход идут переменные report_file и folder_name, осуществляет отбор файлов для обработки и печатает в файл статистику по файлам

razbor_las.py
на вход идет переменная las_file полученная из razbor_files_in_folder.py полная статистика по 1 лас с записью в массив определенного вида

las_stat.py
на вход модули razbor_files_in_folder.py и razbor_las.py формирует общий массив итогов и выводит их в файл
Баги: Плохо с русским текстом

Aplication.py
Графическая оболочка программы операции получения входной и выходной папки


Баги:

1)Взаимодействие файлов с разными кодировками неотлажено, предпочтительные символы ASCII
2)Программа находится в препарированном состоянии (необходимо дополнительное обеспечение по кодировкам)
